# iPhone使用手順

## 1. GitHub Pages公開

`Settings` → `Pages` → `Build and deployment` → `Branch` を以下に設定します。

- Branch: `main`
- Folder: `/root`

保存後、GitHub PagesのURLが表示されます。

## 2. iPhoneホーム画面追加

1. iPhoneのSafariで公開URLを開く
2. 共有ボタンを押す
3. `ホーム画面に追加`
4. 名前を `現調台帳` にして追加

## 3. 現調時の基本操作

1. 現場基本情報を入力
2. 現場条件チェックを入力
3. 測定台帳で `面積追加`、`延長追加`、`個数追加` を使い分ける
4. 劣化箇所を追加
5. 写真枠を追加してiPhoneカメラで撮影
6. 保存
7. 必要に応じてPDF保存・テキスト出力・JSONバックアップ

## 4. 測定入力ルール

### 面積

幅・長さ × 高さ・奥行 × 数量 − 控除 = ㎡

例：外壁 6.0 × 2.8 × 1 − 1.8 = 15.00㎡

### 延長

幅・長さ × 数量 − 控除 = m

例：シーリング 8.0 × 3 − 0 = 24.00m

### 個数

数量 − 控除 = 箇所

例：鉄部蓋 6 − 0 = 6箇所

## 5. データ保存注意

入力内容と写真はiPhone内のブラウザ保存です。
写真を大量に入れると容量超過する場合があります。
重要案件は必ず以下を併用してください。

- PDF保存
- JSONバックアップ
- テキスト出力

## 6. 推奨運用

写真は「外壁南面」「屋上ドレン」「鉄部階段」など、部位ごとに写真枠を分けてください。
測定根拠には「レーザー実測」「スケール実測」「控除あり」「再確認必要」などを残すと見積時に強いです。
