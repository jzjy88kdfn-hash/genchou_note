# genchou_note

現調時にiPhoneで使うための、単体HTML型の現調台帳アプリです。

## 目的

現場調査時に、写真・スケール実測・レーザー測定・劣化メモ・見積前確認事項を1画面で整理します。

## すぐ使う方法

GitHub Pagesを有効化してください。

1. GitHubで `genchou_note` を開く
2. `Settings` を開く
3. `Pages` を開く
4. `Build and deployment` の `Branch` を `main`、フォルダを `/root` にする
5. `Save`
6. 表示されたURLをiPhone Safariで開く
7. 共有ボタン → ホーム画面に追加

## 主な機能

- 現場基本情報
- 現場条件チェック
- 面積・延長・個数の測定台帳
- 自動合計
- 劣化・施工メモ
- iPhoneカメラ写真登録
- 見積前整理
- 最終判定
- 端末内保存
- JSONバックアップ
- JSON復元
- テキスト出力
- 共有
- PDF保存
- ホーム画面追加対応
- オフラインキャッシュ

## 保存仕様

入力内容と写真はブラウザのローカル保存を使います。外部サーバーへ送信しません。

写真を大量に入れると保存容量を超える場合があります。重要案件はPDF保存またはJSONバックアップを併用してください。

## 現場での推奨運用

1. 現調前にホーム画面から起動確認
2. 現場名・住所・元請を入力
3. 測定ごとに面積・延長・個数を分けて入力
4. 写真は部位ごとに写真枠を分ける
5. 劣化は部位・症状・緊急度を残す
6. 最後に保存、PDF、JSONバックアップ
