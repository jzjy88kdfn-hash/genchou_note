# genchou_note

iPhone現調用の、現場別・項目別・実測集計型の現調台帳アプリです。

## 現在の仕様

現場名を登録し、現場ごとに項目を作成して、実測値を連続入力します。現調結果では項目別に数量が自動集計されます。

内訳書への転記・見積書作成機能はありません。現調データの整理と数量確認に特化しています。

## 基本の流れ

1. 現場名を登録
2. 現場を選択
3. 項目名と単位を登録
4. 実測開始
5. 登録項目から選択、またはその場で項目作成
6. 数量を入力
7. `登録して続けて入力` で同一項目を連続入力
8. 現調結果で項目別合計を確認
9. 必要に応じてCSV、PDF、JSONバックアップ

## 対応単位

- ㎡
- m
- 箇所
- 枚
- 式

## 数量入力

### ㎡

縦・高さ × 横・幅 × 数量 − 開口控除 = 数量

開口控除は複数追加できます。

### m

長さ × 数量 − 控除m = 数量

控除メモも残せます。

### 箇所・枚・式

数量入力のみです。

## 写真

画像は任意です。登録してもしなくても実測保存できます。

写真は長辺1600px目安でJPEG圧縮してブラウザ内に保存します。

## 保存仕様

入力内容と写真はブラウザのローカル保存を使います。外部サーバーへ送信しません。

重要案件では、必ず以下を併用してください。

- 選択現場JSONバックアップ
- 全現場JSONバックアップ
- CSV出力
- PDF保存

## GitHub Pagesで使う方法

1. GitHubで `genchou_note` を開く
2. `Settings` を開く
3. `Pages` を開く
4. `Build and deployment` の `Branch` を `main`、フォルダを `/root` にする
5. `Save`
6. 表示されたURLをiPhone Safariで開く
7. 共有ボタン → ホーム画面に追加

## iPhoneで古い画面が出る場合

Safariでページを再読込してください。それでも古い場合は、ホーム画面アイコンを一度削除して、Safariから再度ホーム画面に追加してください。
